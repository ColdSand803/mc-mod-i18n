# Quick Start

## Minimal workflow

1. Choose an input type in the workspace, such as `Mod JAR`.
2. Select the source language and target language.
3. Choose a translation provider.
4. Click `Run preflight` to confirm the input can be processed.
5. Click `Start generation` after the preflight result looks acceptable.

## Suggestions

- For a first run, use `glossary` or `deep-free` to verify the workflow.
- If preflight reports blocking issues, fix those before starting generation.
- After generation, use the result page, translation report, and task history to inspect the output.
