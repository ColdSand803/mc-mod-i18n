# 翻译器选择

## 零配置试用

- `deep-free`
  - 无需 API Key
  - 适合快速试跑
  - 稳定性受第三方公共源影响

## 离线或本地能力

- `glossary`
  - 完全离线
  - 只做规则替换
- `argos`
  - 本地离线翻译
  - 需要安装依赖和语言包

## 可控免费方案

- `libretranslate`
  - 适合自托管或固定实例
  - 比公共免费网页源更可控

## 官方稳定方案

- `azure-translator`
  - 需要 API Key
  - 某些资源类型需要 `Region`

## 通用 AI 接口

- `openai-compatible`
- `anthropic-compatible`

适合追求更高质量，但通常有成本。
