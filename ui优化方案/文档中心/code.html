快速开始

最小流程

按照以下步骤，快速完成第一次完整的翻译流程：

  1. 在 工作区 选择输入类型，例如 Mod JAR。
  2. 选择源语言和目标语言（确保方向正确）。
  3. 选择一个可用的 翻译器。
  4. 点击 运行预检，先确认输入格式和设置可处理。
  5. 预检通过后，再次点击 开始生成。

lightbulb 建议与提示

  • 第一次使用可以先用 glossary 或 deep-free 跑通流程，验证环境配置。
  • 如果预检出现 阻断 (Error)，请优先修复阻断项，不要直接忽略警告强制运行。
  • 处理完成后，可在 结果页、翻译报告 或 任务历史 里继续排查可能的遗漏。
